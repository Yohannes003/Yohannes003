## 📡 About Me

I'm **Yohannes**, a senior full-stack developer based in **Addis Ababa, Ethiopia**, working in a government-adjacent security context at **INSA Information Network Security Agency**.

I design and build production systems at the intersection of **software engineering, data intelligence, automation and applied AI**. My work includes distributed backend services, multi-tenant platforms, real-time dashboards, OSINT/SOCMINT collection pipelines, machine-learning workflows and full-stack web and mobile applications.

* 🛰️ Building **OSINT and SOCMINT systems** that transform raw open-source information into searchable, structured and actionable intelligence
* 🕷️ Developing resilient **web-scraping and browser-automation pipelines** using Playwright, Selenium, Beautiful Soup and distributed workers
* ⚙️ Automating data collection, validation, enrichment, deduplication, scheduling and monitoring workflows
* 🧠 Building applied AI systems with **PyTorch, Hugging Face Transformers, LangChain, RAG and vector databases**
* 🏷️ Experienced in **AI response evaluation, data annotation, labeling, rubric design and quality assurance**
* 🧪 Evaluating model outputs for factuality, instruction following, relevance, reasoning quality and practical usefulness
* 🏗️ Designing scalable **FastAPI, Django and microservice architectures** backed by PostgreSQL, Redis, Kafka and Docker
* 📱 Shipping end-to-end products across backend, frontend and mobile using React, Next.js and Flutter
* 📊 Building analytics platforms, live dashboards, alerting systems and ML-powered monitoring tools
* ♟️ Chess player following a tactics-first improvement track
* 🇫🇷 Learning French with an interest in francophone African technology markets
* 🌍 Building toward senior and staff-level remote engineering opportunities
* 📫 Reach me at **[yohannes.g003@gmail.com](mailto:yohannes.g003@gmail.com)**
